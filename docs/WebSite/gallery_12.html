Heating coil

This visualization uses various plots to show the geometry and flow induced by heating of a coil inside a cylinder with solution. The input dataset is an unstructured mesh (mixed element types) stored in CGNS format. The streamlines show the flow through the solution.

The dataset was provided by the CGNS user community's Example files page. Click here to download the dataset.
